倪妮个人简介

基本信息

倪妮，1988年8月8日生于南京，毕业于中国传媒大学南广学院语言传播系，中国内地影视女演员。

2011年，因出演战争史诗电影《金陵十三钗》女主角玉墨一角而进入演艺圈，并凭借此片获得第六届亚洲电影大奖最佳新演员等多个奖项 [1]。2013年1月，与Angelababy、杨幂及刘诗诗被评为内地新“四小花旦” [2]，随后主演爱情悬疑电影《杀戒》、爱情电影《我想和你好好的》及治愈系电影《等风来》。2014年，主演校园爱情电影《匆匆那年》，同年成为第九届中国作家榜授予的首任“阅读星大使”。2015年，主演浪漫爱情喜剧电影《新娘大作战》。2016年，主演奇幻动作冒险电影《勇士之门》和爱情片《28岁未成年》。2017年，主演奇幻电影《悟空传》和《奇门遁甲》。2018年，主演的古装权谋剧《天盛长歌》于8月14日登陆湖南卫视金鹰独播剧场，犯罪电影《雪暴》也即将上映。2019年11月，凭借电视剧《宸汐缘》获得第26届华鼎奖中国古装题材电视剧最佳女演员的提名。

早年经历

1988年8月8日出生于南京的倪妮，小学就读于南京长江路小学，初中考入南京九中，高中就读于南京市梅园中学，高三转到江苏省盱眙中学。2007年考入中国传媒大学南广学院，于2011年6月毕业。在校期间她是国家二级游泳运动员，还获得过江苏省国际标准舞冠军。

个人生活

2012年5月24日，冯绍峰发表微博，公开承认与倪妮恋情。

2015年5月17日，有媒体曝倪妮与冯绍峰分手，两人恋情于近日正式划下句点，双方都已各向挚友透露恢复单身。

2016年3月，倪妮和井柏然恋情爆光，双方经纪人承认恋情。

2018年7月5日，井柏然工作室官方发布声明，宣布井柏然与倪妮于五月已经和平分手。
